# 👻 SnapVault — Zero Persistence App

SnapVault is a cloud-based ephemeral messaging system inspired by Snapchat.

## 🚀 Features
- Send snaps with message + emoji
- Auto-delete after 5 minutes (TTL)
- Serverless architecture
- Real-time retrieval

## 🏗️ Architecture
Browser (S3) → API Gateway → Lambda → DynamoDB

## 🛠️ Technologies
- AWS S3
- AWS Lambda (Node.js)
- API Gateway
- DynamoDB (TTL)
